Movie Rating Analysis - Hex Softwares Task 2

Project Overview
This project analyzes the IMDb Top 1000 movies dataset to uncover trends in ratings, genres, and financial performance.

Key Features
- Data Cleaning: Handled missing values in Gross earnings and Meta-scores.
- Statistical Analysis: Calculated mean, median, and mode for ratings.
- Visualizations: 
  - Histogram of rating distributions.
  - Box plot to identify rating outliers.
  - Bar charts for genre frequency and popularity.

Conclusion
The analysis reveals that Drama and Comedy are the most frequent genres, while the top-rated films are often concentrated in the 7.6 to 8.2 rating range.
